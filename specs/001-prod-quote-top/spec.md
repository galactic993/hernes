# フィーチャ仕様 — 制作見積書作成<トップ画面>

> `design-doc-reader` の出力イメージ（実設計書からの抜粋を写経）。出典シートを併記。

## サマリ

制作部署が、営業からの見積依頼を含む制作見積情報を検索・一覧表示し、詳細確認・得意先選択を経て
制作見積作成へ進むための画面。（出典: 画面概要シート）

## ユーザー / 権限

（出典: 画面概要「利用権限 一覧」）
- 制作部門: 所長 / チームリーダー / マニュアルエディター / 翻訳コーディネーター / システムエンジニア / Webディレクター
- システム管理者
- 権限識別子: `editorial.prod-quote.create`（出典: イベント記述書 EVENT0001）

## ゴール / 非ゴール

- ゴール: 未着手の制作見積を検索・一覧し、作成・詳細へ遷移できる
- 非ゴール: 制作見積登録・更新（02-02_3）、制作見積依頼（02-02_4）

## 機能要件（出典: 項目記述書 / イベント記述書）

- FR-001: 初期表示時、ログインユーザー所属センターが主管の未着手の制作見積を取得し一覧表示する（EVENT0001 / 参照仕様）
- FR-002: 制作見積情報検索（ステータス・見積書No・得意先・見積件名・品名）で絞り込める（EVENT0009 / 項目記述書 No.1,2,3,5,6）
- FR-003: 一覧行から見積書No・選択で詳細/作成へ遷移できる（EVENT0010, EVENT0011）
- FR-004: 得意先選択モーダルで得意先コード/得意先名で検索し選択できる（EVENT0012〜0016 / 項目記述書 No.16,17,20）
- FR-005: 見積情報詳細・制作見積内容詳細をモーダル表示する（EVENT0018, EVENT0019）

## 非機能要件

- NFR-001: 検索フローはログ/メトリクスで観測可能（個人情報は出さない）

## セキュリティ要件（出典: イベント記述書 / メッセージ一覧）

- SEC-001: `editorial.prod-quote.create` を持たないユーザーは「アクセス権限がありません」を表示しポータルへ遷移（メッセージ一覧 No.1）

## バリデーション（出典: 項目記述書 制御内容 → メッセージ）

| 項目No | 項目 | 制御内容 | メッセージ |
|---|---|---|---|
| 1 | ステータス | 必須(初期値あり) / 有効な値のみ | ステータスは入力必須です / 有効な値を選択してください |
| 2 | 見積書No | 任意 / 11桁以内 | 11桁以内で入力してください |
| 16 | 得意先コード | 任意 / 半角数字のみ / 5桁以内 | 半角数字で入力してください / 5桁以内で入力してください |
| 20 | 得意先選択 | 必須 / 有効な値のみ | 得意先を選択してください / 有効な値を選択してください |

→ `@hernes/shared`（`validation/prod-quote-search.ts`, `messages.ts`, `status.ts`）に単一定義。

## エラー/サクセスメッセージ（出典: メッセージ一覧）

- No.1 権限なし: アクセス権限がありません
- No.3 検索 値不正: 入力内容に誤りがあります。各項目をご確認ください
- No.6 検索結果なし（サクセス）: 該当する制作見積情報が見つかりません
- No.10 得意先選択 値不正: 得意先を選択してください

## データ（出典: 編-テーブル定義書 / 制作見積シート）

  → `@hernes/db/src/schema/prod-quots.ts`
